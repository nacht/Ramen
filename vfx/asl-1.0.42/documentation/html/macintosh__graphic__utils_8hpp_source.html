00028 /*
00029     WARNING (fbrereto) : Due to the way Mac OS X reference counts (in this case) CGImageRef objects,

00031                          trying to delete the cg_image_t. The issue here is the state information

00036                          the CGImageRef that it creates, otherwise you'll be diving off a bad
00037                          pointer.
00038 */
